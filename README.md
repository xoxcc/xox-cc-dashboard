# XOX CC Call Activity Dashboard

Internal contact centre dashboard for XOX CC team.

## Setup
1. Clone this repo
2. Enable GitHub Pages: Settings → Pages → Branch: `main` → Folder: `/ (root)`
3. Access at `https://<your-username>.github.io/<repo-name>/`

## Features
- KPI Summary (Total, Answered, Abandoned, SLA)
- Daily Call Volume Trend
- Hourly Call Volume
- Agent Performance & Call Drop
- Incident Reports
- User Management (Admin)
- Dark / Light mode
- Import 3CX call reports (CSV/Excel)
