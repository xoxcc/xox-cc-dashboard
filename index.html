User Management
How to Give Dashboard Access
Step 1 — Run this SQL in Supabase SQL Editor to seed all staff at once:
INSERT INTO public.cc_users (id, email, name, role, permissions, active) SELECT id, email, split_part(email,'@',1), 'viewer', '{"kpi":true,"trend":true,"hourly":true,"sla":true,"agent_cards":true,"agent_table":true,"incidents":true}'::jsonb, true FROM auth.users WHERE email LIKE '%@xox.com.my' ON CONFLICT (id) DO NOTHING;
Step 2 — Staff log in with their email & password. They appear below automatically.
Step 3 — Click any name below to edit it. Change role and permissions as needed.
To reset ALL staff passwords to Xox@2026 at once:
UPDATE auth.users SET encrypted_password = crypt('Xox@2026', gen_salt('bf')), updated_at = now() WHERE email LIKE '%@xox.com.my';
